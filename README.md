# Karangula Sathwik — Data Scientist & Analyst Portfolio

A high-performance, dynamic portfolio website showcasing machine learning, computer vision, and data analysis capabilities with interactive simulations and structured layout architecture.

## 🚀 Live Site
Once deployed via GitHub Actions, the live website will be available at:
**[https://satu2423.github.io/Portfolio/](https://satu2423.github.io/Portfolio/)**

---

## ✦ Key Features

1. **Aesthetic & Theme Controls**: Sleek editorial typography mixing Sans-serif, Serif, and Monospace typefaces with Dark/Light theme toggle controls.
2. **Interactive Data Science Playground**:
   - **Predictive Forecasting**: Adjust marketing spend, active leads, and seasonality to view quarterly projections dynamically calculated using client-side linear regression coefficients.
   - **Computer Vision Specialist Tool**: Toggle image preprocessing steps (Gaussian Blur, Canny Edge Detection, Bounding Box Annotations) to simulate machine learning annotation pipelines.
3. **Structured Architecture Layer Stack**: Mapped architecture of competencies ranging from mathematical foundations to cloud-deployed generative intelligence.
4. **Automated Contact Form Integration**: Direct, responsive contact submissions powered by FormSubmit.co, delivering messages directly to `sathwikkarangula2423@gmail.com`.
5. **Real-time Clock (IST)**: Active status pill synchronizing local Hyderabad time.

---

## 🛠 Tech Stack

- **Core**: HTML5, Vanilla JavaScript (ES Modules)
- **Styling**: Custom CSS (using OKLCH colors, fluid typography, and premium ease transitions)
- **Tooling/Bundler**: Vite
- **Continuous Deployment**: GitHub Actions & GitHub Pages

---

## 💻 Local Development

### 1. Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed (v18 or higher recommended).

### 2. Install Dependencies
```bash
npm install
```

### 3. Run Development Server
```bash
npm run dev
```

### 4. Build for Production
```bash
npm run build
```
